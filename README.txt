Interactive photo browser, or Photothèque. 
Allows the user to view photos and create 
annotations in the form of text and strokes.

Code is very neat:
- MVC design pattern fully implemented in Java Swing
- Commentary when needed

Features:
- Open photo (File -> Open)
- Delete photo (File -> Delete)
- Flip photo (double click)
- Draw strokes (click and drag mouse)
- Draw text, wrapped when needed (click and type)

Extra features:
- The drawn color can be set (Draw -> Set Color)
- The text font can be set too (Draw -> Set Font)
